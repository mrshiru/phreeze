Installation Instructions:
--------------------------

1. Copy "classbuilder" to your web root

2. copy "scripts" to your web root

3. Copy "includes" to any location within your PHP include_path


Troubleshooting:
----------------

If you are unable to copy files to your web root, you may put classbuilder and scripts into
a subdirectory.  Please note that when you do this, you will need to override the 
"parameters" settings when generating applications so that your generated application
will use the correct path to the "Scripts" folder.

For additional information, please visit the project home page at http://www.phreeze.com/
